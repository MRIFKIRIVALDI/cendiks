Ada 3 metode untuk memasang Python di Windows. Pilih yang paling sesuai untuk Anda!

Metode 1: Lewat Microsoft Store (Paling Mudah)

Langkah-langkah:

  1. Buka Microsoft Store
  2. Cari: "Python"
  3. Pilih versi terbaru (biasanya Python 3.x)
  4. Klik Get / Install
  5. Tunggu hingga proses selesai

Verifikasi Instalasi:

Untuk mengecek instalasi, buka Command Prompt dan ketik:

python --version

Metode 3: Menggunakan Windows Package Manager (winget)

Cocok untuk pengguna Windows 10/11 yang suka command line.

  1. Buka PowerShell
  2. Jalankan perintah:
winget install Python.Python.3
  1. Tunggu proses instalasi

Cek versi:

python --version

üí° Tips Penting:

  • Pastikan untuk mencentang "Add Python to PATH" saat instalasi metode 2
  • Jika ada error, coba jalankan Command Prompt sebagai Administrator
  • Untuk uninstall, gunakan Settings > Apps > Python > Uninstall
  • Python akan otomatis tersedia di Command Prompt setelah instalasi
