Instalasi Python di Linux

Ubuntu/Debian/Fedora/Arch - Pilih distro Linux Anda!

Metode 1: Ubuntu/Debian (apt)

Langkah-langkah:

  1. Buka Terminal
  2. Update repositori:
sudo apt update
  1. Install Python:
sudo apt install python3 python3-pip
  1. Cek versi:
python3 --version
üî∂

Metode 2: Fedora/RHEL (dnf)

Langkah-langkah:

  1. Buka Terminal
  2. Jalankan perintah install:
sudo dnf install python3 python3-pip
  1. Cek versi:
python3 --version
üî∑

Metode 4: Arch Linux/Manjaro (pacman)

Langkah-langkah:

  1. Buka Terminal
  2. Jalankan perintah install:
sudo pacman -S python python-pip
  1. Cek versi:
python --version
üß∞

Metode 4: Build dari Source Code (Advanced)

Dipakai jika ingin versi terbaru atau custom build.

Download source Python:

wget https://www.python.org/ftp/python/3.x.x/Python-3.x.x.tgz

Ekstrak:

tar -xvzf Python-3.x.x.tgz

Build:

./configure make sudo make install
