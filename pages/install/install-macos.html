Metode 1: Lewat Homebrew (Paling Direkomendasikan untuk macOS)

Cocok buat developer atau pengguna aktif terminal.

Instal Homebrew (jika belum):

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Install Python:

brew install python

Cek versi:

python3 --version

Metode 2: Download dari python.org (Paling Mudah untuk Pemula)

Langkah-langkah:

  1. Buka: https://www.python.org/downloads/mac-osx/
  2. Download file .pkg
  3. Buka file tersebut ‚Üí installer akan berjalan
  4. Klik Continue sampai selesai

Cek instalasi:

python3 --version

Metode 3: Instal via Xcode Command Line Tools

Cara alternatif yang banyak dipakai developer macOS.

  1. Buka Terminal
  2. Jalankan:
xcode-select --install

macOS menyediakan Python versi bawaan (meski biasanya lebih lama)

Cek dengan:

python3 --version
