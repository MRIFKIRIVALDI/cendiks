Pelajari cara memanipulasi list dalam Python

Materi Pembelajaran

1Ô∏è‚É£ Append

Menambahkan elemen di akhir list.

my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  # Output: [1, 2, 3, 4]

2Ô∏è‚É£ Insert

Menambahkan elemen di posisi tertentu.

my_list = [1, 2, 3]
my_list.insert(1, 'a')
print(my_list)  # Output: [1, 'a', 2, 3]

3Ô∏è‚É£ Remove

Menghapus elemen berdasarkan nilai.

my_list = [1, 2, 3, 2]
my_list.remove(2)
print(my_list)  # Output: [1, 3, 2]

4Ô∏è‚É£ Pop

Menghapus elemen berdasarkan indeks.

my_list = [1, 2, 3]
removed = my_list.pop(1)
print(my_list)  # Output: [1, 3]
print(removed)  # Output: 2

5Ô∏è‚É£ Clear

Menghapus semua elemen.

my_list = [1, 2, 3]
my_list.clear()
print(my_list)  # Output: []

6Ô∏è‚É£ Sort

Mengurutkan list.

my_list = [3, 1, 4, 1, 5]
my_list.sort()
print(my_list)  # Output: [1, 1, 3, 4, 5]

7Ô∏è‚É£ Reverse

Membalik urutan list.

my_list = [1, 2, 3, 4]
my_list.reverse()
print(my_list)  # Output: [4, 3, 2, 1]

8Ô∏è‚É£ Contoh Penggunaan

Program yang mengelola daftar tugas.

tasks = []
while True:
    print("1. Tambah tugas")
    print("2. Lihat tugas")
    print("3. Hapus tugas")
    print("4. Keluar")
    choice = input("Pilih: ")
    if choice == '1':
        task = input("Tugas: ")
        tasks.append(task)
    elif choice == '2':
        for i, task in enumerate(tasks):
            print(f"{i+1}. {task}")
    elif choice == '3':
        index = int(input("Nomor tugas: ")) - 1
        if 0 <= index < len(tasks):
            tasks.pop(index)
    elif choice == '4':
        break
