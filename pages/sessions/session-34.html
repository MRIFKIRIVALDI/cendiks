Scope: wilayah kekuasaan variabel! üåç Global bisa ke mana-mana, Local cuma di rumahnya sendiri üè†

# Global: bisa diakses dimana saja
nama = "Alice"
def sapa():
    print(f"Halo {nama}")
sapa()  # Halo Alice

# Local: cuma di dalam fungsi
def hitung():
    x = 10
    return x * 2
print(hitung())  # 20

2Ô∏è‚É£ Global Scope

Global: variabel duniawi yang bisa dipanggil dari mana saja! üåç‚ú®

nama = "Alice"

def sapa():
    print(f"Halo {nama}")

sapa()  # Halo Alice

def ubah():
    global nama
    nama = "Bob"

ubah()
print(nama)  # Bob

3Ô∏è‚É£ Local Scope

Variabel yang didefinisikan di dalam fungsi hanya dapat diakses dari dalam fungsi tersebut.

# Contoh Local Scope
def fungsi_local():
    local_var = "Saya local"
    print(local_var)  # Output: Saya local

fungsi_local()
# print(local_var)  # Error: name 'local_var' is not defined

# Nested functions dan scope
def outer():
    outer_var = "outer"

    def inner():
        inner_var = "inner"
        print(outer_var)  # Bisa mengakses variabel outer
        print(inner_var)  # Bisa mengakses variabel inner

    inner()
    # print(inner_var)  # Error: tidak bisa mengakses dari outer

outer()

5Ô∏è‚É£ nonlocal Keyword

nonlocal: jembatan untuk mengubah variabel di scope luar! üåâ‚ú®

def outer():
    x = "awal"

    def inner():
        nonlocal x
        x = "diubah"
        print(f"Inner: {x}")

    inner()
    print(f"Outer: {x}")

outer()
# Inner: diubah
# Outer: diubah

6Ô∏è‚É£ Latihan Praktis

Ayo latihan scope dengan contoh simpel! üí™

# Counter global
counter = 0

def tambah():
    global counter
    counter += 1
    return counter

print(tambah())  # 1
print(counter)   # 1

# Calculator dengan nonlocal
def buat_kalkulator():
    hasil = 0

    def tambah(x):
        nonlocal hasil
        hasil += x
        return hasil

    return tambah

tambah = buat_kalkulator()
print(tambah(5))  # 5
print(tambah(3))  # 8
