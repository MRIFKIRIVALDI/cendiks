Operator logika digunakan untuk menggabungkan kondisi boolean.

  • and: Kedua kondisi harus True
  • or: Salah satu kondisi harus True
  • not: Membalik nilai boolean
x = 5
y = 10
print(x > 3 and y < 15)  # True
print(x > 3 or y > 15)   # True
print(not(x > 3))        # False
