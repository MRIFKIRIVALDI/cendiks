Pelajari operasi-operasi pada list dan cara menyalin list

1Ô∏è‚É£ Concatenation

Menggabungkan list dengan operator +.

list1 = [1, 2, 3]
list2 = [4, 5, 6]
result = list1 + list2
print(result)  # Output: [1, 2, 3, 4, 5, 6]

2Ô∏è‚É£ Repetition

Mengulang list dengan operator *.

my_list = [1, 2]
result = my_list * 3
print(result)  # Output: [1, 2, 1, 2, 1, 2]

3Ô∏è‚É£ Membership

Mengecek apakah elemen ada di list dengan operator in.

my_list = [1, 2, 3, 4, 5]
print(3 in my_list)  # Output: True
print(6 in my_list)  # Output: False

4Ô∏è‚É£ Copy List

Menyalin list agar tidak reference.

original = [1, 2, 3]
copy1 = original[:]  # Shallow copy dengan slicing
copy2 = original.copy()  # Shallow copy dengan method

5Ô∏è‚É£ Shallow Copy

Copy tingkat atas, nested list masih reference.

import copy
original = [[1, 2], [3, 4]]
shallow = copy.copy(original)
shallow[0][0] = 99
print(original)  # Output: [[99, 2], [3, 4]]

6Ô∏è‚É£ Deep Copy

Copy semua level dengan deepcopy().

import copy
original = [[1, 2], [3, 4]]
deep = copy.deepcopy(original)
deep[0][0] = 99
print(original)  # Output: [[1, 2], [3, 4]]

7Ô∏è‚É£ Contoh Penggunaan

Program yang menggabungkan dan menyalin list.

def merge_lists(*lists):
    result = []
    for lst in lists:
        result += lst  # Concatenation
    return result

list1 = [1, 2, 3]
list2 = [4, 5, 6]
merged = merge_lists(list1, list2)
print(merged)  # Output: [1, 2, 3, 4, 5, 6]
