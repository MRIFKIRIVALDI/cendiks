Pelajari cara melakukan looping pada dictionary

Materi Pembelajaran

1Ô∏è‚É£ Looping Keys

Melakukan iterasi pada keys dictionary.

person = {"nama": "John", "umur": 25, "kota": "Jakarta"}

# Cara 1: Loop langsung pada dictionary (default loop keys)
for key in person:
    print(key)

# Cara 2: Menggunakan keys()
for key in person.keys():
    print(key)

# Output:
# nama
# umur
# kota

2Ô∏è‚É£ Looping Values

Melakukan iterasi pada values dictionary.

person = {"nama": "John", "umur": 25, "kota": "Jakarta"}

for value in person.values():
    print(value)

# Output:
# John
# 25
# Jakarta

3Ô∏è‚É£ Looping Items

Melakukan iterasi pada key-value pairs.

person = {"nama": "John", "umur": 25, "kota": "Jakarta"}

for key, value in person.items():
    print(f"{key}: {value}")

# Output:
# nama: John
# umur: 25
# kota: Jakarta

4Ô∏è‚É£ Nested Dictionary

Dictionary di dalam dictionary.

mahasiswa = {
    "alice": {"umur": 20, "jurusan": "Informatika"},
    "bob": {"umur": 21, "jurusan": "Sistem Informasi"},
    "charlie": {"umur": 19, "jurusan": "Teknik Komputer"}
}

# Loop pada nested dictionary
for nama, data in mahasiswa.items():
    print(f"Nama: {nama}")
    for key, value in data.items():
        print(f"  {key}: {value}")
    print()

5Ô∏è‚É£ Iterasi Nested

Looping dalam nested dictionary.

company = {
    "IT": {
        "manager": "John",
        "employees": ["Alice", "Bob", "Charlie"]
    },
    "HR": {
        "manager": "Sarah",
        "employees": ["David", "Eva"]
    }
}

for department, data in company.items():
    print(f"Department: {department}")
    print(f"Manager: {data['manager']}")
    print("Employees:")
    for employee in data['employees']:
        print(f"  - {employee}")
    print()

6Ô∏è‚É£ Contoh Penggunaan

Program yang menampilkan data dari dictionary.

def tampilkan_profil(profil):
    """Fungsi untuk menampilkan profil pengguna"""
    for key, value in profil.items():
        if isinstance(value, dict):
            print(f"{key.upper()}:")
            for sub_key, sub_value in value.items():
                print(f"  {sub_key}: {sub_value}")
        else:
            print(f"{key}: {value}")
    print("-" * 30)

# Data profil
profil = {
    "nama": "John Doe",
    "umur": 30,
    "kontak": {
        "email": "john@example.com",
        "telepon": "08123456789"
    },
    "pekerjaan": "Software Engineer"
}

tampilkan_profil(profil)
