Pelajari operator perbandingan untuk membandingkan nilai dalam Python

Materi Pembelajaran

1Ô∏è‚É£ Pengenalan Operasi Komparasi

Operasi komparasi (perbandingan) digunakan untuk membandingkan dua nilai dan menghasilkan nilai boolean (True atau False). Ini sangat penting dalam pengambilan keputusan dalam program.

# Contoh sederhana operasi komparasi
a = 5
b = 10

print(a == b)  # False - apakah a sama dengan b?
print(a != b)  # True  - apakah a tidak sama dengan b?
print(a < b)   # True  - apakah a lebih kecil dari b?
print(a > b)   # False - apakah a lebih besar dari b?

2Ô∏è‚É£ Operator Sama Dengan (==)

Operator == digunakan untuk memeriksa apakah dua nilai sama. Perlu diingat bahwa == berbeda dengan = (yang digunakan untuk assignment).

# Contoh penggunaan ==
x = 5
y = 5
z = 10

print(x == y)     # True - x sama dengan y
print(x == z)     # False - x tidak sama dengan z

# Dengan string
nama1 = "Alice"
nama2 = "Bob"
nama3 = "Alice"

print(nama1 == nama3)  # True - string sama
print(nama1 == nama2)  # False - string berbeda

3Ô∏è‚É£ Operator Tidak Sama Dengan (!=)

Operator != digunakan untuk memeriksa apakah dua nilai tidak sama. Ini adalah kebalikan dari operator ==.

# Contoh penggunaan !=
a = 100
b = 200
c = 100

print(a != b)     # True - a tidak sama dengan b
print(a != c)     # False - a sama dengan c

# Dengan boolean
status1 = True
status2 = False

print(status1 != status2)  # True - True tidak sama dengan False
print(status1 != True)     # False - True sama dengan True

4Ô∏è‚É£ Operator Lebih Besar (>) dan Lebih Kecil (<)

Operator > dan < digunakan untuk membandingkan nilai numerik. > untuk lebih besar, < untuk lebih kecil.

# Contoh penggunaan > dan <
nilai1 = 85
nilai2 = 92
nilai3 = 78

print(nilai2 > nilai1)   # True - 92 > 85
print(nilai3 > nilai1)   # False - 78 > 85
print(nilai1 < nilai2)   # True - 85 < 92
print(nilai2 < nilai3)   # False - 92 < 78

# Dengan float
harga1 = 15.50
harga2 = 20.75

print(harga2 > harga1)   # True - 20.75 > 15.50
print(harga1 < harga2)   # True - 15.50 < 20.75

5Ô∏è‚É£ Operator Lebih Besar Sama Dengan (>=) dan Lebih Kecil Sama Dengan (<=)

Operator >= dan <= adalah kombinasi dari > dan ==, serta < dan ==. Mereka memeriksa apakah nilai lebih besar/kecil atau sama dengan.

# Contoh penggunaan >= dan <=
umur = 18
batas_umur = 18

print(umur >= batas_umur)  # True - 18 >= 18 (boleh ikut pemilu)

# Contoh nilai ujian
nilai = 75
kkm = 70

print(nilai >= kkm)        # True - lulus
print(nilai <= 100)        # True - nilai valid

# Dengan variabel yang sama
x = 10
print(x >= x)              # True - 10 >= 10
print(x <= x)              # True - 10 <= 10

6Ô∏è‚É£ Membandingkan String

String juga dapat dibandingkan menggunakan operator komparasi. Perbandingan dilakukan berdasarkan urutan alfabet (lexicographical order).

# Contoh membandingkan string
kata1 = "Apple"
kata2 = "Banana"
kata3 = "Apple"

print(kata1 == kata3)      # True - sama persis
print(kata1 != kata2)      # True - berbeda
print(kata1 < kata2)       # True - 'A' sebelum 'B' dalam alfabet
print(kata2 > kata1)       # True - 'B' setelah 'A' dalam alfabet

# Case sensitive
nama1 = "alice"
nama2 = "Alice"

print(nama1 == nama2)      # False - huruf kecil vs kapital
print(nama1 < nama2)       # True - 'a' (97) < 'A' (65) dalam ASCII

7Ô∏è‚É£ Menggunakan Komparasi dalam Kondisi

Operasi komparasi sering digunakan dalam statement if, while, dan lainnya untuk mengontrol alur program.

# Contoh penggunaan dalam if
nilai_ujian = 85

if nilai_ujian >= 80:
    print("Selamat! Anda lulus dengan nilai A")
elif nilai_ujian >= 70:
    print("Anda lulus dengan nilai B")
else:
    print("Anda perlu mengulang")

# Contoh dengan while
counter = 0
while counter <= 5:
    print(f"Counter: {counter}")
    counter += 1

# Contoh validasi input
umur = 25
if umur >= 18 and umur <= 65:
    print("Umur valid untuk bekerja")
else:
    print("Umur tidak memenuhi syarat")

8Ô∏è‚É£ Latihan Praktis

Buat program sederhana yang menggunakan operasi komparasi untuk menentukan grade nilai siswa:

# Program penentuan grade
def tentukan_grade(nilai):
    if nilai >= 90:
        return "A"
    elif nilai >= 80:
        return "B"
    elif nilai >= 70:
        return "C"
    elif nilai >= 60:
        return "D"
    else:
        return "E"

# Test program
nilai_siswa = [95, 87, 76, 62, 45]

for nilai in nilai_siswa:
    grade = tentukan_grade(nilai)
    print(f"Nilai {nilai} = Grade {grade}")

# Program cek bilangan
def cek_bilangan(angka):
    if angka > 0:
        print(f"{angka} adalah bilangan positif")
    elif angka < 0:
        print(f"{angka} adalah bilangan negatif")
    else:
        print(f"{angka} adalah nol")

cek_bilangan(10)
cek_bilangan(-5)
cek_bilangan(0)
