Pelajari struktur data Dictionary dan Set untuk menyimpan data yang lebih kompleks

Materi Pembelajaran

1Ô∏è‚É£ Apa itu Dictionary?

Dictionary adalah struktur data yang menyimpan pasangan key-value. Dictionary menggunakan kurung kurawal {} dan setiap item dipisahkan oleh koma.

# Membuat dictionary
mahasiswa = {
    "nama": "John",
    "umur": 20,
    "jurusan": "Informatika"
}

# Mengakses nilai
print(mahasiswa["nama"])  # Output: John

2Ô∏è‚É£ Operasi Dictionary

Berikut adalah operasi-operasi yang dapat dilakukan pada dictionary:

  • Menambah item: dict[key] = value
  • Mengubah nilai: dict[key] = new_value
  • Menghapus item: del dict[key]
  • Mengecek keberadaan key: key in dict
# Contoh operasi
mahasiswa["nilai"] = 85  # Menambah item
mahasiswa["umur"] = 21   # Mengubah nilai
del mahasiswa["jurusan"] # Menghapus item

print("nama" in mahasiswa)  # Output: True

3Ô∏è‚É£ Method Dictionary

Dictionary memiliki beberapa method yang berguna:

  • keys() - Mengembalikan semua key
  • values() - Mengembalikan semua nilai
  • items() - Mengembalikan pasangan key-value
  • get(key, default) - Mengakses nilai dengan aman
print(mahasiswa.keys())    # dict_keys(['nama', 'umur', 'nilai'])
print(mahasiswa.values())  # dict_values(['John', 21, 85])
print(mahasiswa.items())   # dict_items([('nama', 'John'), ('umur', 21), ('nilai', 85)])

# Menggunakan get() untuk menghindari error
print(mahasiswa.get("alamat", "Tidak ada"))  # Output: Tidak ada

4Ô∏è‚É£ Apa itu Set?

Set adalah struktur data yang menyimpan elemen unik tanpa urutan. Set menggunakan kurung kurawal {} atau fungsi set().

# Membuat set
buah = {"apel", "jeruk", "mangga"}
angka = set([1, 2, 3, 4, 5])

print(buah)   # Output: {'apel', 'jeruk', 'mangga'}
print(angka)  # Output: {1, 2, 3, 4, 5}

5Ô∏è‚É£ Operasi Set

Operasi-operasi yang dapat dilakukan pada set:

  • Menambah elemen: add()
  • Menghapus elemen: remove() atau discard()
  • Union: | atau union()
  • Intersection: & atau intersection()
  • Difference: - atau difference()
set1 = {1, 2, 3}
set2 = {3, 4, 5}

set1.add(6)           # {1, 2, 3, 6}
set1.remove(2)        # {1, 3, 6}

print(set1 | set2)    # Union: {1, 3, 4, 5, 6}
print(set1 & set2)    # Intersection: {3}
print(set1 - set2)    # Difference: {1, 6}

6Ô∏è‚É£ Latihan Praktis

Buat program yang menggunakan dictionary dan set:

# Dictionary untuk menyimpan data mahasiswa
mahasiswa = {
    "nama": "Alice",
    "nim": "12345",
    "mata_kuliah": {"Pemrograman", "Matematika", "Fisika"}
}

print(f"Nama: {mahasiswa['nama']}")
print(f"NIM: {mahasiswa['nim']}")
print(f"Mata Kuliah: {mahasiswa['mata_kuliah']}")

# Menambah mata kuliah baru
mahasiswa["mata_kuliah"].add("Kimia")
print(f"Mata Kuliah setelah ditambah: {mahasiswa['mata_kuliah']}")
