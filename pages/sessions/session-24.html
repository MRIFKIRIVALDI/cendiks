Pelajari dictionary dalam Python

Materi Pembelajaran

1Ô∏è‚É£ Apa Itu Dictionary?

Koleksi data dengan key-value pair yang tidak terurut.

# Membuat dictionary
my_dict = {
    "nama": "John",
    "umur": 25,
    "kota": "Jakarta"
}

print(my_dict)  # Output: {'nama': 'John', 'umur': 25, 'kota': 'Jakarta'}

2Ô∏è‚É£ Membuat Dictionary

Berbagai cara membuat dictionary.

# Dengan kurung kurawal
dict1 = {"nama": "Alice", "umur": 30}

# Dengan dict() constructor
dict2 = dict(nama="Bob", umur=25)

# Dari list of tuples
pairs = [("nama", "Charlie"), ("umur", 35)]
dict3 = dict(pairs)

print(dict1)  # Output: {'nama': 'Alice', 'umur': 30}
print(dict2)  # Output: {'nama': 'Bob', 'umur': 25}
print(dict3)  # Output: {'nama': 'Charlie', 'umur': 35}

3Ô∏è‚É£ Key dan Value

Key unik, value bisa apa saja.

my_dict = {
    "string": "hello",
    42: "number as key",
    ("tuple", "key"): "tuple as key",
    "list": [1, 2, 3]  # list sebagai value
}

print(my_dict["string"])      # Output: hello
print(my_dict[42])            # Output: number as key
print(my_dict[("tuple", "key")])  # Output: tuple as key

4Ô∏è‚É£ Mengakses Value

Menggunakan key untuk mengakses value.

person = {"nama": "John", "umur": 25, "kota": "Jakarta"}

print(person["nama"])   # Output: John
print(person["umur"])   # Output: 25

# Menggunakan get() method (lebih aman)
print(person.get("kota"))     # Output: Jakarta
print(person.get("alamat", "Tidak ada"))  # Output: Tidak ada

5Ô∏è‚É£ Update Dictionary

Menambahkan atau mengubah key-value.

person = {"nama": "John", "umur": 25}

# Menambah key baru
person["pekerjaan"] = "Programmer"
print(person)  # Output: {'nama': 'John', 'umur': 25, 'pekerjaan': 'Programmer'}

# Mengubah value
person["umur"] = 26
print(person)  # Output: {'nama': 'John', 'umur': 26, 'pekerjaan': 'Programmer'}

# Menggunakan update()
person.update({"kota": "Bandung", "hobi": "Coding"})
print(person)  # Output: {'nama': 'John', 'umur': 26, 'pekerjaan': 'Programmer', 'kota': 'Bandung', 'hobi': 'Coding'}

6Ô∏è‚É£ Delete Key

Menghapus key dari dictionary.

person = {"nama": "John", "umur": 25, "kota": "Jakarta"}

# Menggunakan del
del person["kota"]
print(person)  # Output: {'nama': 'John', 'umur': 25}

# Menggunakan pop()
umur = person.pop("umur")
print(person)  # Output: {'nama': 'John'}
print(umur)    # Output: 25

# Menggunakan popitem() (hapus item terakhir)
person["pekerjaan"] = "Programmer"
removed = person.popitem()
print(person)  # Output: {'nama': 'John'}
print(removed) # Output: ('pekerjaan', 'Programmer')

7Ô∏è‚É£ Method Dictionary

keys(), values(), items(), dll.

person = {"nama": "John", "umur": 25, "kota": "Jakarta"}

print(person.keys())    # Output: dict_keys(['nama', 'umur', 'kota'])
print(person.values())  # Output: dict_values(['John', 25, 'Jakarta'])
print(person.items())   # Output: dict_items([('nama', 'John'), ('umur', 25), ('kota', 'Jakarta')])

# Iterasi
for key in person.keys():
    print(key)

for value in person.values():
    print(value)

for key, value in person.items():
    print(f"{key}: {value}")

8Ô∏è‚É£ Contoh Penggunaan

Program yang menyimpan data mahasiswa.

mahasiswa = {
    "nama": "Alice",
    "nim": "123456",
    "jurusan": "Informatika",
    "nilai": {
        "matematika": 85,
        "fisika": 90,
        "kimia": 88
    }
}

print(f"Nama: {mahasiswa['nama']}")
print(f"NIM: {mahasiswa['nim']}")
print(f"Nilai Matematika: {mahasiswa['nilai']['matematika']}")

# Menghitung rata-rata
nilai = mahasiswa['nilai']
rata_rata = sum(nilai.values()) / len(nilai)
print(f"Rata-rata: {rata_rata}")
