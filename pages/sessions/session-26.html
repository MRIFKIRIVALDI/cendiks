Pelajari dasar-dasar fungsi dalam Python

Materi Pembelajaran

1Ô∏è‚É£ Apa Itu Fungsi?

Fungsi adalah blok kode yang dapat dipanggil berulang kali untuk melakukan tugas tertentu.

def sapa():
    print("Halo, selamat datang!")

# Memanggil fungsi
sapa()  # Output: Halo, selamat datang!
sapa()  # Output: Halo, selamat datang!

2Ô∏è‚É£ Mengapa Menggunakan Fungsi?

Meningkatkan reusability kode, memudahkan debugging, membuat kode lebih terorganisir.

# Tanpa fungsi - kode berulang
print("Selamat pagi, John!")
print("Apakah Anda siap belajar?")
print("Selamat pagi, Alice!")
print("Apakah Anda siap belajar?")

# Dengan fungsi - lebih efisien
def sapa_pengguna(nama):
    print(f"Selamat pagi, {nama}!")
    print("Apakah Anda siap belajar?")

sapa_pengguna("John")
sapa_pengguna("Alice")

3Ô∏è‚É£ Mendefinisikan Fungsi

Sintaks dasar untuk membuat fungsi.

def nama_fungsi():
    # blok kode
    statement1
    statement2
    # ...

# Contoh
def cetak_info():
    print("Ini adalah fungsi sederhana")
    print("Fungsi ini mencetak informasi")

cetak_info()

4Ô∏è‚É£ Memanggil Fungsi

Cara menjalankan fungsi yang telah didefinisikan.

def hitung_luas_persegi(sisi):
    luas = sisi * sisi
    print(f"Luas persegi dengan sisi {sisi} adalah {luas}")

# Memanggil fungsi
hitung_luas_persegi(5)   # Output: Luas persegi dengan sisi 5 adalah 25
hitung_luas_persegi(10)  # Output: Luas persegi dengan sisi 10 adalah 100

5Ô∏è‚É£ Parameter dan Argument

Parameter dalam definisi, argument saat pemanggilan.

# Fungsi dengan parameter
def sapa(nama):  # 'nama' adalah parameter
    print(f"Halo, {nama}!")

# Memanggil dengan argument
sapa("John")    # "John" adalah argument
sapa("Alice")   # "Alice" adalah argument

# Fungsi dengan multiple parameter
def perkenalan(nama, umur):
    print(f"Nama saya {nama}, umur saya {umur} tahun")

perkenalan("Bob", 25)

6Ô∏è‚É£ Return Value

Mengembalikan nilai dari fungsi.

def tambah(a, b):
    hasil = a + b
    return hasil

# Menggunakan nilai return
result1 = tambah(5, 3)
result2 = tambah(10, 7)

print(result1)  # Output: 8
print(result2)  # Output: 17

# Return dapat digunakan langsung
print(tambah(2, 4))  # Output: 6

7Ô∏è‚É£ Scope Variabel

Local dan global scope.

# Global variable
nama = "John"

def sapa():
    # Local variable
    salam = "Halo"
    print(f"{salam}, {nama}!")

sapa()  # Output: Halo, John!

# print(salam)  # Error: name 'salam' is not defined (local scope)

def ubah_nama():
    global nama  # Mengakses global variable
    nama = "Alice"

ubah_nama()
print(nama)  # Output: Alice

8Ô∏è‚É£ Contoh Penggunaan

Fungsi untuk menghitung luas persegi.

def hitung_luas_persegi_panjang(panjang, lebar):
    """Fungsi untuk menghitung luas persegi panjang"""
    return panjang * lebar

def hitung_keliling_persegi_panjang(panjang, lebar):
    """Fungsi untuk menghitung keliling persegi panjang"""
    return 2 * (panjang + lebar)

# Menggunakan fungsi
p = 10
l = 5

luas = hitung_luas_persegi_panjang(p, l)
keliling = hitung_keliling_persegi_panjang(p, l)

print(f"Luas: {luas}")       # Output: Luas: 50
print(f"Keliling: {keliling}")  # Output: Keliling: 30
