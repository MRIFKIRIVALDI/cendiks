Type casting adalah proses mengubah tipe data dari satu jenis ke jenis lainnya.

2Ô∏è‚É£ Implicit Casting

Python secara otomatis mengubah tipe data, misalnya int ke float.

3Ô∏è‚É£ Explicit Casting

Programmer secara manual mengubah tipe data menggunakan fungsi seperti int(), float(), str().

4Ô∏è‚É£ Casting ke Integer

Menggunakan int(), misalnya int("123") menjadi 123.

5Ô∏è‚É£ Casting ke Float

Menggunakan float(), misalnya float("3.14") menjadi 3.14.

6Ô∏è‚É£ Casting ke String

Menggunakan str(), misalnya str(123) menjadi "123".

7Ô∏è‚É£ Casting ke Boolean

Menggunakan bool(), nilai non-zero menjadi True, zero menjadi False.

8Ô∏è‚É£ Contoh Penggunaan

Contoh program yang menggunakan casting untuk input user.
