1Ô∏è‚É£ Apa Itu Pip?

Pip adalah package manager resmi untuk Python yang digunakan untuk menginstall, upgrade, dan mengelola packages dari Python Package Index (PyPI).

# Cek versi pip
pip --version

# Update pip ke versi terbaru
python -m pip install --upgrade pip

# Install package
pip install requests

# Install package dengan versi spesifik
pip install requests==2.28.0

# Install dari file requirements.txt
pip install -r requirements.txt

# Uninstall package
pip uninstall requests

# List packages yang terinstall
pip list

# Show informasi package
pip show requests

# Search package di PyPI
pip search beautifulsoup4

# Upgrade package
pip install --upgrade requests

# Install package dalam development mode
pip install -e .

# Install dari Git repository
pip install git+https://github.com/user/repo.git

# Install dengan dependencies minimal
pip install --no-deps package-name

2Ô∏è‚É£ Virtual Environments

Menggunakan virtual environment untuk mengisolasi dependencies project.

# Membuat virtual environment
python -m venv myproject

# Aktivasi virtual environment (Windows)
myproject\Scripts\activate

# Aktivasi virtual environment (macOS/Linux)
source myproject/bin/activate

# Deaktivasi virtual environment
deactivate

# Install packages dalam virtual environment
pip install requests flask

# Membuat requirements.txt
pip freeze > requirements.txt

# Install dari requirements.txt
pip install -r requirements.txt

# Menggunakan pipenv (alternatif modern)
pip install pipenv

# Membuat virtual environment dengan pipenv
pipenv install

# Install package dengan pipenv
pipenv install requests

# Install dev dependencies
pipenv install --dev pytest

# Activate pipenv shell
pipenv shell

# Jalankan script dalam pipenv
pipenv run python app.py

# Menggunakan conda (untuk data science)
conda create -n myenv python=3.9
conda activate myenv
conda install numpy pandas

3Ô∏è‚É£ Requirements.txt dan Dependency Management

Mengelola dependencies project dengan requirements.txt dan best practices.

# Contoh requirements.txt
requests==2.28.0
flask==2.2.2
pandas>=1.5.0
numpy>=1.21.0
matplotlib
scikit-learn==1.1.3

# Dengan komentar
# Web framework
flask==2.2.2
# HTTP requests
requests==2.28.0

# Install dari requirements.txt
pip install -r requirements.txt

# Membuat requirements.txt dari environment
pip freeze > requirements.txt

# Hanya packages yang diinstall secara eksplisit
pip list --format=freeze > requirements.txt

# Menggunakan pip-tools untuk dependency locking
pip install pip-tools

# Membuat requirements.in
echo "requests" > requirements.in
echo "flask" >> requirements.in

# Generate requirements.txt dengan versi terbaru
pip-compile requirements.in

# Upgrade semua dependencies
pip-compile --upgrade requirements.in

# Install dari compiled requirements
pip install -r requirements.txt

# Menggunakan Poetry (modern dependency management)
pip install poetry

# Inisialisasi project
poetry init

# Add dependencies
poetry add requests flask

# Add dev dependencies
poetry add --group dev pytest black

# Install dependencies
poetry install

# Jalankan command dalam poetry shell
poetry run python app.py

4Ô∏è‚É£ Troubleshooting dan Common Issues

Mengatasi masalah umum saat menggunakan Pip.

# Error: pip is not recognized
# Solusi: Pastikan Python sudah terinstall dengan benar
python -m pip --version

# Error: Permission denied
# Solusi: Gunakan --user flag atau jalankan sebagai admin
pip install --user package-name

# Error: Package not found
# Solusi: Cek nama package yang benar
pip search package-name
pip install package-name

# Error: SSL certificate verify failed
# Solusi: Disable SSL verification (tidak recommended untuk production)
pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org package-name

# Error: Microsoft Visual C++ 14.0 is required
# Solusi: Install Build Tools atau gunakan pre-compiled wheels
pip install --only-binary=all package-name

# Error: pip install --upgrade pip gagal
# Solusi: Gunakan python -m pip
python -m pip install --upgrade pip

# Error: ModuleNotFoundError setelah install
# Solusi: Pastikan menggunakan virtual environment yang benar
# atau install dengan --user flag

# Error: Conflicting dependencies
# Solusi: Gunakan pip-tools atau Poetry untuk dependency resolution
pip install pip-tools
pip-compile requirements.in

# Error: Slow download
# Solusi: Gunakan mirror atau cache
pip install --cache-dir /path/to/cache package-name
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple package-name

# Error: Disk space full
# Solusi: Bersihkan pip cache
pip cache purge

# Error: ImportError setelah upgrade
# Solusi: Reinstall package atau gunakan versi yang kompatibel
pip install --force-reinstall --no-deps package-name

# Error: pip freeze shows too many packages
# Solusi: Gunakan pip list --format=freeze untuk packages yang diinstall eksplisit
pip list --format=freeze > requirements.txt

5Ô∏è‚É£ Latihan Praktis

Buat project Python dengan dependency management yang baik.

# 1. Buat direktori project
mkdir my-python-project
cd my-python-project

# 2. Buat virtual environment
python -m venv venv

# 3. Aktivasi virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 4. Install dependencies awal
pip install requests flask

# 5. Buat requirements.txt
pip freeze > requirements.txt

# 6. Buat file setup.py untuk package
echo 'from setuptools import setup, find_packages

setup(
    name="my-python-project",
    version="0.1.0",
    packages=find_packages(),
    install_requires=[
        "requests>=2.28.0",
        "flask>=2.2.0",
    ],
)' > setup.py

# 7. Install package dalam development mode
pip install -e .

# 8. Buat file .gitignore
echo '__pycache__/
*.pyc
*.pyo
*.pyd
.Python
env/
venv/
.venv/
pip-log.txt
pip-delete-this-directory.txt
.coverage
.pytest_cache/
' > .gitignore

# 9. Menggunakan pip-tools untuk dependency management
pip install pip-tools
echo 'requests
flask
pytest' > requirements.in
pip-compile requirements.in
pip install -r requirements.txt
