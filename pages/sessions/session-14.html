Pelajari cara bekerja dengan tanggal dan waktu dalam Python

Materi Pembelajaran

1Ô∏è‚É£ Module datetime

Module datetime menyediakan kelas untuk bekerja dengan tanggal dan waktu.

from datetime import datetime, date, time

# Tanggal dan waktu saat ini
sekarang = datetime.now()
print(sekarang)

2Ô∏è‚É£ Membuat Objek DateTime

Cara membuat objek datetime dari komponen tanggal dan waktu.

from datetime import datetime

# Membuat datetime spesifik
dt = datetime(2023, 12, 25, 10, 30, 45)
print(dt)

3Ô∏è‚É£ Formatting DateTime

Mengubah objek datetime menjadi string dengan format tertentu.

from datetime import datetime

sekarang = datetime.now()
formatted = sekarang.strftime("%Y-%m-%d %H:%M:%S")
print(formatted)

4Ô∏è‚É£ Parsing String ke DateTime

Mengubah string menjadi objek datetime.

from datetime import datetime

date_string = "2023-12-25 10:30:45"
dt = datetime.strptime(date_string, "%Y-%m-%d %H:%M:%S")
print(dt)

5Ô∏è‚É£ Operasi DateTime

Menambah atau mengurangi waktu dari datetime.

from datetime import datetime, timedelta

sekarang = datetime.now()
besok = sekarang + timedelta(days=1)
kemarin = sekarang - timedelta(days=1)
print("Besok:", besok)
print("Kemarin:", kemarin)
