Pelajari tuple dan set dalam Python

Materi Pembelajaran

1ï¸âƒ£ Tuple

Koleksi data terurut yang tidak dapat diubah (immutable).

# Membuat tuple
my_tuple = (1, 2, 3, "hello", 3.14)
empty_tuple = ()
single_tuple = (1,)  # Perlu koma untuk tuple dengan satu elemen

print(my_tuple)     # Output: (1, 2, 3, 'hello', 3.14)
print(type(my_tuple))  # Output:

2ï¸âƒ£ Membuat Tuple

Berbagai cara membuat tuple.

# Dengan tanda kurung
tuple1 = (1, 2, 3)

# Tanpa tanda kurung (tuple packing)
tuple2 = 1, 2, 3

# Dari list
my_list = [1, 2, 3]
tuple3 = tuple(my_list)

print(tuple1)  # Output: (1, 2, 3)
print(tuple2)  # Output: (1, 2, 3)
print(tuple3)  # Output: (1, 2, 3)

3ï¸âƒ£ Set

Koleksi data unik yang tidak terurut.

# Membuat set
my_set = {1, 2, 3, 4, 5}
empty_set = set()  # {} membuat dict kosong, bukan set

print(my_set)      # Output: {1, 2, 3, 4, 5}
print(type(my_set)) # Output:

4ï¸âƒ£ Membuat Set

Berbagai cara membuat set.

# Dengan kurung kurawal
set1 = {1, 2, 3}

# Dari list (menghilangkan duplikat)
my_list = [1, 2, 2, 3, 3, 3]
set2 = set(my_list)

# Dari string
set3 = set("hello")

print(set1)  # Output: {1, 2, 3}
print(set2)  # Output: {1, 2, 3}
print(set3)  # Output: {'l', 'o', 'e', 'h'}

5ï¸âƒ£ Perbedaan Tuple vs List

Tuple immutable, list mutable.

# List - dapat diubah
my_list = [1, 2, 3]
my_list[0] = 99
print(my_list)  # Output: [99, 2, 3]

# Tuple - tidak dapat diubah
my_tuple = (1, 2, 3)
# my_tuple[0] = 99  # Akan error: TypeError

6ï¸âƒ£ Perbedaan Set vs List

Set unik dan tidak terurut.

# List mempertahankan urutan dan duplikat
my_list = [1, 2, 2, 3, 1]
print(my_list)  # Output: [1, 2, 2, 3, 1]

# Set menghilangkan duplikat dan tidak terurut
my_set = {1, 2, 2, 3, 1}
print(my_set)   # Output: {1, 2, 3} (urutan mungkin berbeda)

7ï¸âƒ£ Method Tuple

count() dan index().

my_tuple = (1, 2, 3, 2, 4, 2)

print(my_tuple.count(2))  # Output: 3 (jumlah kemunculan 2)
print(my_tuple.index(3))  # Output: 2 (indeks pertama 3)

8ï¸âƒ£ Method Set

add(), remove(), union(), dll.

my_set = {1, 2, 3}

# Menambah elemen
my_set.add(4)
print(my_set)  # Output: {1, 2, 3, 4}

# Menghapus elemen
my_set.remove(2)
print(my_set)  # Output: {1, 3, 4}

# Operasi himpunan
set1 = {1, 2, 3}
set2 = {3, 4, 5}
print(set1.union(set2))      # Output: {1, 2, 3, 4, 5}
print(set1.intersection(set2))  # Output: {3}

9ï¸âƒ£ Contoh Penggunaan

Program yang menggunakan tuple dan set.

# Tuple untuk data yang tidak berubah
coordinates = (10.5, 20.3)
person = ("John", 25, "Engineer")

# Set untuk menghilangkan duplikat
emails = ["a@example.com", "b@example.com", "a@example.com"]
unique_emails = set(emails)
print(unique_emails)  # Output: {'a@example.com', 'b@example.com'}
